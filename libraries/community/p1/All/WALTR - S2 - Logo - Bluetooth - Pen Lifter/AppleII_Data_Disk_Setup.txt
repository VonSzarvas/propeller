Setting up Apple Logo II data.dsk.
----------------------------------

In directory "AppleLogoII_LCSI_1984" rename file WALTR to WALTR.TXT
Open CiderPress
Open data.dsk (ProDOS disk)
Below the :LOGO line right click and select "Add Files"
In Text conversion tick "Convert text files by file type"
Select "WALTR.txt"
Press the "Accept" button.
Right click on the added file and select "Rename"
Rename "WALTR.TXT" to "WALTR"

Do the same with the HOUSE file.


File Format details :-

File System:	ProDOS 1.0.1
Space		0x20
New Line	0x0D
File Type 	TXT 0x04
Aux type	0x0000
