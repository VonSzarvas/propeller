FSRW currently boasts the following features:

  • Read and write speeds in excess of 900KB/s (raw) and 600KB/s (through the file system layer); significantly faster when using a 100MHz Propeller
  • Compatibility with Secure Digital, Secure Digital High Capacity, and MMC cards
  • Support for FAT16 and FAT32
  • Multiblock support for extra speed
  • Read, write, and append modes
  • Character I/O, memory block I/O, and string output
  • Directory listing
